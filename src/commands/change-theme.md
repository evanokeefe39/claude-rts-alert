---
name: rts-alert:change-theme
description: Change the active RTS sound theme
---
<objective>
Switch to a different sound theme without re-installing hooks.
</objective>

<process>
Run the following command:

```
npx claude-rts-alert setup
```

The setup wizard will let you pick a new theme and preview its sounds. Since all themes are already installed, switching is instant — only the config file is updated.
</process>
