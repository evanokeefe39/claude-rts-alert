Frequency/pattern scheme:
- **wc3-orc** (low/bass tones): base 200Hz
  - task-complete: rising two-note (200Hz -> 300Hz), 0.3s -- triumphant
  - needs-input: repeated pulse (250Hz x3), 0.5s -- attention-getting

  - error: descending (300Hz -> 150Hz), 0.3s -- negative feedback

- **wc3-human** (mid tones): base 400Hz
  - task-complete: rising two-note (400Hz -> 600Hz), 0.3s
  - needs-input: repeated pulse (450Hz x3), 0.5s
  - greeting: single tone (400Hz), 0.4s
  - error: descending (600Hz -> 300Hz), 0.3s

- **aoe2** (mid-high tones): base 600Hz
  - task-complete: rising two-note (600Hz -> 900Hz), 0.3s
  - needs-input: repeated pulse (650Hz x3), 0.5s
  - greeting: single tone (600Hz), 0.4s
  - error: descending (900Hz -> 450Hz), 0.3s

- **classic-windows** (high tones): base 800Hz
  - task-complete: rising two-note (800Hz -> 1200Hz), 0.3s
  - needs-input: repeated pulse (850Hz x3), 0.5s
  - greeting: single tone (800Hz), 0.4s
  - error: descending (1200Hz -> 600Hz), 0.3s

WAV generation: Write raw PCM data with a proper RIFF/WAVE header. 8-bit mono, 8000Hz sample rate. Use sine wave generation with Math.sin(2 * PI * freq * t / sampleRate). For multi-note patterns, split the buffer into segments.

   Import THEMES, THEME_NAMES, getThemeSoundFile from '../themes'.

   ```typescript
   describe('theme sound file resolution (integration)', () => {
     const events: HookEvent[] = ['stop', 'notification', 'session-start', 'post-tool-use'];

     for (const theme of THEME_NAMES) {
       for (const event of events) {
         it(`${theme}/${event} resolves to existing WAV file`, () => {
           const soundFile = getThemeSoundFile(theme, event);
           const absPath = resolveAssetPath(soundFile);
           expect(fs.existsSync(absPath)).toBe(true);
           const stat = fs.statSync(absPath);
           expect(stat.size).toBeGreaterThan(44); // Valid WAV header minimum
         });
       }
     }

     it('covers all 16 theme-event combinations', () => {
       expect(THEME_NAMES.length * events.length).toBe(16);
     });
   });
   ```
