    - uninstall() deletes ~/.claude-notify/ config directory recursively if it exists
    - uninstall() prints confirmation message to stdout

    1. Create `src/cli/uninstall.ts`:
       - Import readSettings, writeSettings from `../config`
       - Import path, fs, os
